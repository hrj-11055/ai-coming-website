部署到服务器的文件清单
==============================================
需要上传的文件和目录:
✓ package.json

✓ .env.example
✓ server-json.js
✓ run.sh
✓ deploy-ubuntu.sh
✓ *.html (index, news, admin-*)

✓ config/

不需要上传的:
✗ node_modules/ (服务器上npm install)
✗ data/ (服务器上自动创建)
✗ .env (服务器上手动创建)
✗ backup/ (删除)
✗ test-* (删除)
✗ *.md (或移至docs/)
✗ server.js (删除)
✗ 旧版 API 适配脚本 (已删除)

==============================================
快速清理命令
==============================================
# 删除备份和测试文件
rm -rf backup/ test-* debug.html *.py qwen-comparison-*.json

# 删除弃用服务器
rm -f server.js

# 删除冗余文件
rm -f requirements.txt
rm -f data/news-2026-01-16.json data/news-upload-example.json

# 移动文档
mkdir -p docs
mv *.md docs/ 2>/dev/null
mv README.md CLAUDE.md PROJECT_CLEANUP_PLAN.md . 2>/dev/null
