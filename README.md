# 动态热点词云系统 - 简化版

## 系统概述

这是一个完全可配置的动态热点词云系统，**简化版**让您只需要提供按重要性排序的30个关键词列表，系统会自动根据顺序分配大小和权重，生成美观的词云效果。

## 🎯 核心特性

### ✨ 极简配置
- **只需提供关键词列表**：按重要性排序的文本数组
- **自动分配大小**：系统根据顺序自动分配字体大小和权重
- **智能排序**：权重越高，字体越大，位置越居中

### 📊 自动分配规则
- **前5个**：超大尺寸，权重10-9（顶级热点）
- **第6-15个**：大尺寸，权重8-6（高热度）
- **第16-25个**：中尺寸，权重5-3（重要热点）
- **第26-30个**：小尺寸，权重2-1（一般热点）

## 使用方法

### 方法一：直接修改配置文件（推荐）

1. 打开 `keywords-config.js` 文件
2. 修改 `HOT_KEYWORDS_CONFIG.keywords` 数组中的关键词列表
3. 保存文件
4. 刷新网站即可看到新效果

**示例：**
```javascript
keywords: [
    'GPT-5发布',        // 第1名：超大尺寸，权重10
    'Claude 4.0',       // 第2名：超大尺寸，权重9
    'Sam Altman',       // 第3名：超大尺寸，权重8
    'AI Agent',         // 第4名：超大尺寸，权重7
    '多模态模型',        // 第5名：超大尺寸，权重6
    'Firefly视频',      // 第6名：大尺寸，权重8
    '通义千问3',        // 第7名：大尺寸，权重8
    // ... 更多关键词（系统自动分配大小和权重）
]
```

### 方法二：使用简化管理界面（推荐）

1. 打开 `simple-keyword-manager.html` 文件
2. 在界面中添加/编辑/删除关键词
3. 拖拽调整关键词顺序
4. 点击"查看网站"按钮预览效果
5. 使用导出功能保存配置

### 方法三：使用完整管理界面（高级功能）

1. 打开 `keyword-manager.html` 文件
2. 在管理界面中添加/编辑/删除关键词
3. 手动设置大小和权重
4. 点击"查看网站"按钮预览效果

### 方法四：编程方式管理

在浏览器控制台中使用以下命令：

```javascript
// 刷新词云
window.wordCloudManager.refresh();

// 添加关键词（简化版）
window.wordCloudManager.add('新关键词');

// 批量设置关键词
window.wordCloudManager.setKeywords(['关键词1', '关键词2', '关键词3']);

// 删除关键词
window.wordCloudManager.remove('要删除的关键词');

// 获取当前关键词
console.log(window.wordCloudManager.getCurrent());

// 导出数据
console.log(window.wordCloudManager.export());
```

## 权重和排序规则

1. **权重范围**：1-10（数值越大越重要）
2. **排序规则**：按权重降序排列（权重高的在前）
3. **字体大小**：权重越高，字体越大
4. **位置优先级**：权重最高的关键词会尽量居中显示

### 权重建议
- **10分**：顶级热点（如重大发布、重要人物）
- **8-9分**：高热度（如重要产品、技术突破）
- **6-7分**：重要热点（如行业动态、政策法规）
- **4-5分**：一般热点（如工具更新、观点分享）
- **1-3分**：小众话题（如细分领域、新兴概念）

## 尺寸说明

## 常见问题

### Q: 如何添加新的关键词？
A: 在 `keywords-config.js` 中的 `keywords` 数组末尾添加新的关键词对象，或使用管理界面添加。

### Q: 为什么我的关键词没有显示？
A: 请检查：
1. 关键词数据格式是否正确
2. 权重值是否在1-10范围内
3. 浏览器控制台是否有错误信息

### Q: 如何调整关键词的显示顺序？
A: 修改关键词的 `weight` 值，权重越高的关键词排序越靠前。

### Q: 可以添加超过30个关键词吗？
A: 可以，但建议控制在30个以内以保证最佳的视觉效果和性能。

### Q: 如何自定义词云标题？
A: 修改 `keywords-config.js` 中 `config.title` 的值。
